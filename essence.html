Learn CoffeeScript in 2 hours

It took me 2 full days of straight studying to learn the essence of CoffeeScript. Two weeks later, I found myself writing the cleanest, most error-free client-side code I had ever created.

I've set a goal to help bring that initial 2 day learning curve down to 2 fun hours: whether you have done only a little JavaScript programming, or you are already a master.

To capture my learnings, I've developed a few exercises that you can try for yourself. You may be surprised at how fast the essence of CoffeeScript can be absorbed.

The Essence There's no need to declare variables CoffeeScript automatically figures out when a new variable name comes into scope. So don't have to worry about specifying the var. var length = 5; var cost = 9.99; var theme = { textColor: 'green', buttonColor: 'blue' }; length = 5 cost = 9.99 theme = { textColor: 'green', buttonColor: 'blue' } Set some variables to diifferent number, string or boolean values. Use console.log to print them out. true Variables can be used to construct strings Strings defined with double quotes are interpolated. So you can use variables, or any expression really, to construct a string. var itemCount = 4; var cost = 9.99; var summary = "The total for " + itemCount + " items is $" + cost + "."; itemCount = 4 cost = 9.99 summary = "The total for #{itemCount} items is $ #{cost}."
